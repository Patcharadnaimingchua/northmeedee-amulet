ระบบซื้อขายพระเครื่องออนไลน์ ซื้อ ขาย แลกเปลี่ยน พระเครื่องแท้จากทั่วประเทศ

## Features

- สมัครสมาชิก / เข้าสู่ระบบ
- ค้นหาและเลือกซื้อพระเครื่อง
- Wishlist
- ตะกร้าสินค้า
- แจ้งชำระเงิน
- ติดตามคำสั่งซื้อ
- รีวิวสินค้า
- ระบบจัดการหลังบ้าน (Admin)

---

## Tech Stack

- React + Vite
- Node.js + Express
- PostgreSQL
- Prisma ORM
- Docker & Docker Compose
- Cloudinary
